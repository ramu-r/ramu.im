A journalling system I've been building and refining since early 2025. It runs in Obsidian with Claude handling the weekly review and loop maintenance.

The core idea: daily capture requires nothing — write raw lines, answer pre-loaded questions, or just talk through the day. The weekly review is the only intelligence pass. It reads the week, writes a brief, updates the loop files, and pre-loads questions for the coming week.

Loops are tracked intentions — things I'm actively trying to change. Each one has a log, a current-state summary, and a strategy memory section: when a specific situation produced something that worked or blocked, that gets noted and confirmed monthly. When the same blocker shows up again, the review surfaces what worked last time.

The system went through several phases before arriving at the current design. The lesson that drove every major cut: a dumber system that runs beats a smarter one that doesn't.
