## The idea

I've tried a lot of journalling approaches. Free writing. Gratitude lists. Morning pages. They all felt like recording, not thinking. I'd write about what happened but the writing didn't connect to anything — it didn't help me change the things I was trying to change.

What I wanted was different. I had a handful of things I was actively working on — sleep, a habit I was building, something I was learning. These weren't tasks to complete. They were intentions: things I was trying to shift, where the work was showing up consistently, not figuring out what to do once.

The problem with tracking intentions is that history matters more than current state. A habit that was stagnant for a month and then moved is different from one that just started. A blocker I've hit before is different from a new one, because I might already know what works against it. I wanted my system to remember that.

## What I built

The system has two moving parts: loops and a weekly review.

**Loops** are the tracked intentions. Each one is a file that records what's currently understood, a short log of what's happened week to week, and a section for strategy memory — when a particular situation produced something that worked or didn't, that gets noted. It's not a diary entry; it's more like a chart for a specific problem.

**The weekly review** is the intelligence pass. It reads the week's journal entries, updates the loops, and writes a brief — what happened, what patterns are emerging, strategy recall for any blockers that showed up before. Questions for the coming week get pre-loaded into each day's journal file.

Daily journalling has no required structure. Write what happened, answer the pre-loaded questions if they're relevant. A two-line entry counts. The system only infers from volume, so single days don't need to be complete.

The result is a system with a memory. When something's blocking me, the weekly review can surface what worked the last time I hit the same wall. That's the payoff I couldn't get from plain journalling.
